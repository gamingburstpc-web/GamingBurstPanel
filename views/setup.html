Panel Setup Required

No user accounts exist. Create your first admin to get started.

Run this command on your VPS to create your first user:

sudo -u gbpanel gbpanel user add

If running it directly from the installation folder:

sudo -u gbpanel node bin/gbpanel.js user add
  1. SSH into your VPS
    Navigate to the GamingBurst Panel directory.
  2. Run the command above
    Enter a username, password, and choose admin role (y).
  3. Return here and log in
    This page will automatically redirect when a user is created.
