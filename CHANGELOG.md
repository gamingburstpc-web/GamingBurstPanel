# Changelog

All notable changes to the GamingBurst Panel project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to Semantic Versioning.

## [1.0.0] - 2026-07-02
### Added
- Initial release of the GamingBurst Panel!
- Fully responsive, mobile-friendly dark mode interface.
- Core Minecraft server management (Start, Stop, Restart, Console).
- Interactive CLI menu (`gbpanel`) for quick system management.
- Complete File Manager with syntax highlighting and drag-and-drop.
- Built-in multi-user management system with granular role-based permissions.
- Automated installation and one-click update mechanisms.
- Safety-first disk space monitoring to prevent world corruption.
- Playit.gg automated port forwarding integration.
